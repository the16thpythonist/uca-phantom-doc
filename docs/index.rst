Welcome to uca-phantom's documentation!
=======================================

This is the documentation for the `Vision Research Phantom Camera <https://www.phantomhighspeed.com/products>`_ plugin
for the `Libuca camera framework <https://github.com/ufo-kit/libuca>`_. This plugin interfaces all phantom cameras
using the `PH16 <https://confluence.diamond.ac.uk/download/attachments/65899299/v16proto-2.3.pdf?version=1&modificationDate=1500390734000&api=v2>`_
camera protocol.
